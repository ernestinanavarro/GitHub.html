Light Show Project

Orientation Challenge

A complete circuit must have ground(brown wire)and power(red wire)running through the end and begining of the circuit

The circuit on the left right hand corner is not complete becuase there is no power running through.The red wire has to be plugged in the rail with the LED light for the electricty to run through the whole circuit.

Blink Challenge

For this challenge we had to create a code using Python that would have a LED light flash for a certain amount of time. To control the amount of time the LED lights, is by using a time variable shown on the code.

User Input Challenge

For this challenge Ana and I programmed the circuit to make the LED light blink with a touch of a key on a keyboard. The code for the use input is set to if a=='w' meaning if 'w' is clicked then it will run the code.

Lightshow Challenge

The circuit was programmed to have the LED lights blink to a song. We programmed the LED lights to blink to the Peppa Pig Theme Song.

In order to sync the song to the LED lights, we had to use the time variable to set how long the lights would depending on the song. In this challenge in particular, we endured a couple erros. One of the erros was having the code run once when key 'w' is pressed.
